# BoostBank — 个人积分激励工具

> **[⬇ 点击下载最新 APK](release/boostBank.apk?raw=true)**

BoostBank 是一款面向个人使用的安卓应用，用于记录「完成事务获得积分」与「消耗积分兑换奖励」，帮助用户建立可持续的自我激励机制。

## APK 位置

构建完成后，安装包位于：

```
app/build/outputs/apk/debug/boostBank-debug.apk
```

可直接将此 APK 传输到安卓设备上安装。

> 最低支持 Android 8.0（API 26）。

## 构建方式

在项目根目录执行：

```bash
./gradlew assembleDebug
```

## 功能说明

应用底部有 4 个导航页面：**赚**（赚取积分）、**奖**（购买奖励）、**览**（分数概览）、**我**（个人设置）。

### 赚取积分

- 点击右上角「新增事务」创建赚分任务，设置名称、积分值和背景图片
- 点击卡片即可完成一次任务并获得对应积分
- 每张卡片可设置背景图片，通过拖拽和双指缩放调整图片显示位置
- 可开启「完成前确认」防止误触

### 购买奖励

- 点击「新增奖励」创建奖励项目
- 点击卡片即可消耗积分兑换奖励（允许透支为负分）
- 操作方式与赚取积分页面一致

### 分数概览

- 顶部显示当前总积分
- 支持按类型筛选日志：全部 / 获取 / 消耗 / 校准
- 可使用「手动校准」直接修改总积分（支持负数）
- 所有操作均有时间戳记录

### 我的

包含三个功能区域：

**头像**
- 选择图片后进入裁剪界面，可看到完整图片
- 拖拽和双指缩放调整圆形头像框中显示的区域
- 已设置头像后可随时重新裁剪

**通用设置**
- 语言切换：简体中文 / English
- 夜间模式开关
- 赚取积分确认开关
- 兑换奖励确认开关

**背景设置**
- 暖色调背景开关
- 页面背景遮罩透明度调节
- 卡片图片透明度调节
- 为每个页面单独设置背景图片

## 数据存储

- 所有数据保存在本地（Room 数据库 + DataStore），无需网络
- 设置在重启 App 后仍然生效
